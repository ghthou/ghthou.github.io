使用 SiteMesh3 完善页面布局

发表于 2018-01-13 | 分类于 Java

在网页开发中，大部分网页都具有相同的页头，页尾，菜单等模块。 一般情况下我们会将这些共用的代码单独抽取成一个页面，然后进行包含。 虽然这样能够达到代码复用的效果，但是如果引入的页面过多，一来会带来修改不变的效果，二来依然会形成多个页面使用相同的代码 （页面包含代码），此时我们可以使用 SiteMesh3 来妥善解决这个问题
